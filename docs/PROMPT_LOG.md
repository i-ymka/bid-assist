**Формат:** изменение → результат через 1-2 недели → вывод.
**AI-анализы:** ссылки на GitHub Issues с меткой `ai-analysis`.

## Как заполнять

После каждого изменения промпта или настройки:
1. Добавь запись ниже (от новых к старым)
2. Через 1-2 недели заполни "Результат"
3. Когда есть GitHub Issue от AI-анализа — добавь ссылку

```markdown
## YYYY-MM-DD | Аккаунт: yehia/ymka/оба | Win rate до: X% → после: ?%

**Что изменили:** краткое описание (промпт / настройка / код)
**Мотивация:** почему, откуда идея (AI-анализ #N / наблюдение / эксперимент)
**Результат:** — (заполнить через 2 недели)
**Вывод:** — (что оставляем, что откатываем)
```
